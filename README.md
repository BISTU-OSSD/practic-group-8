# 学期课程追踪系统（Semester Course Tracker）

## 项目简介

学期课程追踪系统是一款基于 Python 开发的课程管理工具，用于帮助大学生合理安排学习任务，了解课程学习进度，并根据课程信息生成学习建议。

系统支持导入学校课程表，自动统计课程信息，计算学期进度，并根据课程难度、学分和剩余课程数量推荐学习优先级。

---#

## 项目功能

### 课程导入

- 导入学校课程表（CSV）
- 自动生成课程数据
- 保存为 courses.json

### 课程管理

- 查看所有课程
- 课程信息统计
- 查看剩余课程数量
- 查看课程周数

### 学习推荐

综合以下因素推荐学习顺序：

- 课程学分
- 课程难度
- 剩余课程数量

帮助学生制定更加合理的学习计划。

### 学期统计

提供：

- 当前学期进度
- 已完成课程
- 剩余课程
- 距离考试时间
- 距离学期结束时间

### 数据可视化

显示：

- 学期进度条
- 课程完成情况
- 学习建议

## 使用方法

```bash
git clone https://github.com/你的用户名/你的仓库.git
```

### 2. 进入项目目录

```bash

```

### 3. 运行程序

```bash

```

第一次运行时：

- 导入学校课程CSV
- 自动生成 courses.json

之后即可正常使用所有功能。

## 数据格式

### courses.json

主要字段包括：

- 课程名称
- 教师
- 教室
- 星期
- 节次
- 上课周次
- 学分
- 难度
- 剩余课程数量

### config.json

保存学期配置：

```json
{
    "semester_start": "2026-09-04",
    "semester_end": "2026-12-31"
}
```
